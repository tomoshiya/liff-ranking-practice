## 📋 **LINE Developersでの公開手順**

### **Step 1: LINE Developers Consoleにログイン**

1. https://developers.line.biz/console/ にアクセス
2. LINEアカウントでログイン

---

### **Step 2: プロバイダー・チャネルの確認**

```
プロバイダー: 練習用
チャネル名: ランキングアプリ練習
チャネルタイプ: LINEログイン
```

1. 「ランキングアプリ練習」チャネルをクリック
2. 左メニューから「LIFF」タブを選択

---

### **Step 3: LIFF設定の確認**

#### **現在の設定（確認）**

```
LIFF ID: 2008911809-CBLKsbT1
LIFF URL: https://liff.line.me/2008911809-CBLKsbT1
エンドポイントURL: https://tomoshiya.github.io/liff-ranking-practice/
サイズ: Full
スコープ: 
  - openid
  - profile
  - chat_message.write
```

#### **確認項目**

- [ ] エンドポイントURLが正しいか
- [ ] サイズが「Full」になっているか
- [ ] スコープが3つ設定されているか

---

### **Step 4: テスターの追加**

#### **4-1. 基本設定タブへ移動**

1. 左メニュー「基本設定」をクリック
2. 下にスクロールして「テスター」セクションを探す

#### **4-2. テスターを追加**

```
【追加方法】
1. 「追加」ボタンをクリック
2. テスターのLINE UserIDを入力
3. 「保存」をクリック

【LINE UserIDの取得方法】
テスターに以下を依頼：
1. LINEアプリを開く
2. 設定 → プロフィール
3. 「ID」をコピー
4. あなたに送ってもらう
```

#### **4-3. テスターの上限**

```
⚠️ 注意事項
- テスターモード: 最大50名まで
- 50名を超える場合: 公開申請が必要

【現在の推奨】
- 最初は5-10名程度
- 段階的に増やす
```

---

### **Step 5: プライバシーポリシー・利用規約の登録**

#### **5-1. チャネル基本設定に追加**

1. 左メニュー「基本設定」
2. 下にスクロール「プライバシーポリシーURL」
3. 入力: `https://tomoshiya.github.io/liff-ranking-practice/privacy-policy.html`
4. 「利用規約URL」
5. 入力: `https://tomoshiya.github.io/liff-ranking-practice/terms-of-service.html`
6. 「保存」をクリック

---

### **Step 6: アプリアイコン・説明の設定**

#### **6-1. アプリアイコン（推奨）**

```
サイズ: 512x512 ピクセル
形式: PNG
内容: アプリのロゴ

【暫定対応】
- まだアイコンがない場合はスキップ可能
- 後から追加できる
```

```

```

---

### **Step 7: 公開設定の確認**

#### **7-1. 公開状態を確認**

```
左メニュー「LIFF」→ LIFF設定

【公開状態】
✅ 公開 (Published)
   → テスターは利用可能

❌ 非公開 (Unpublished)
   → まだ誰も利用できない
```

#### **7-2. 公開する（必要に応じて）**

```
1. LIFF設定画面で「公開」ボタンをクリック
2. 確認ダイアログで「OK」

【注意】
⚠️ 「公開」してもテスターモードなら
   テスター登録した人のみが利用可能
```
